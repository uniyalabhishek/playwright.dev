Test retry

Playwright Test will retry tests if they failed. Pass the maximum number of retries when running the tests, or set them in the configuration file.

npx playwright test --retries=3
  • JavaScript
  • TypeScript
// playwright.config.js
module.exports = {
retries: 3,
};
// playwright.config.ts
import { PlaywrightTestConfig } from 'playwright/test';
const config: PlaywrightTestConfig = {
retries: 3,
};
export default config;

Failing tests will be retried multiple times until they pass, or until the maximum number of retries is reached. Playwright Test will report all tests that failed at least once.
